化学反应原理

1. 化学反应与能量

焓变：$ \Delta H = H(生成物) - H(反应物) $

热化学方程式：表示化学反应与反应热关系的化学方程式

盖斯定律：化学反应的反应热只与反应的始态和终态有关，而与反应的途径无关

2. 化学反应速率

定义：$ v = \frac{\Delta c}{\Delta t} $（Δc为浓度变化，Δt为时间变化）

影响因素：浓度、温度、压强、催化剂等

3. 化学平衡

化学平衡常数：对于反应aA + bB ⇌ cC + dD，$ K = \frac{c^c(C)\cdot c^d(D)}{c^a(A)\cdot c^b(B)} $

勒夏特列原理：如果改变影响平衡的一个条件（如浓度、压强或温度），平衡就向能够减弱这种改变的方向移动

4. 弱电解质的电离平衡

电离常数：弱酸$ HA ⇌ H^+ + A^- $，$ K_a = \frac{c(H^+)\cdot c(A^-)}{c(HA)} $

弱碱$ BOH ⇌ B^+ + OH^- $，$ K_b = \frac{c(B^+)\cdot c(OH^-)}{c(BOH)} $

典型例题

例1（基础题）：已知反应2H₂(g) + O₂(g) = 2H₂O(l)的ΔH = -571.6 kJ/mol，求1 mol H₂完全燃烧生成液态水时放出的热量。

解析：根据热化学方程式，2 mol H₂完全燃烧生成液态水时放出的热量为571.6 kJ，所以1 mol H₂完全燃烧生成液态水时放出的热量为571.6 kJ / 2 = 285.8 kJ。

例2（中档题）：在2L的密闭容器中，发生反应3A(g) + B(g) ⇌ 2C(g)，若最初加入的A和B都是4 mol，10 s后，测得v(A) = 0.12 mol/(L·s)，求10 s时容器中B的物质的量。

解析：v(A) = 0.12 mol/(L·s)，则10 s内A的浓度变化Δc(A) = v(A) × Δt = 0.12 mol/(L·s) × 10 s = 1.2 mol/L。根据化学方程式，Δc(B) = Δc(A) / 3 = 0.4 mol/L，所以10 s时B的浓度c(B) = 4 mol / 2L - 0.4 mol/L = 2 mol/L - 0.4 mol/L = 1.6 mol/L，B的物质的量n(B) = c(B) × V = 1.6 mol/L × 2L = 3.2 mol。

例3（提高题）：对于反应CO(g) + H₂O(g) ⇌ CO₂(g) + H₂(g)，在800℃时，平衡常数K = 1。若在该温度下，向一个2L的密闭容器中充入1 mol CO和1 mol H₂O，求平衡时各物质的浓度。

解析：设平衡时CO的浓度变化为x mol/L，则：

CO(g) + H₂O(g) ⇌ CO₂(g) + H₂(g)

起始浓度（mol/L）：0.5 0.5 0 0

变化浓度（mol/L）：x x x x

平衡浓度（mol/L）：0.5 - x 0.5 - x x x

平衡常数K = [CO₂][H₂] / [CO][H₂O] = x² / (0.5 - x)² = 1，解得x = 0.25 mol/L。所以平衡时各物质的浓度为：[CO] = [H₂O] = 0.5 - 0.25 = 0.25 mol/L，[CO₂] = [H₂] = 0.25 mol/L。

常见错误分析
  • 错误1：混淆焓变的符号。
    正确做法：焓变ΔH为负，表示放热反应；ΔH为正，表示吸热反应。
  • 错误2：在计算化学反应速率时，忽略了体积的影响。
    正确做法：化学反应速率的定义是单位时间内浓度的变化，而不是物质的量的变化，所以需要除以体积。
  • 错误3：在使用化学平衡常数时，代入了固体或纯液体的浓度。
    正确做法：化学平衡常数表达式中，只包括气态物质和溶液的浓度，固体和纯液体的浓度视为1，不写入表达式中。
解题技巧总结
  • 对于热化学方程式问题，要注意焓变的符号、单位和化学计量数的关系。
  • 计算化学反应速率时，要注意浓度变化和时间的关系，以及不同物质的速率之比等于化学计量数之比。
  • 使用化学平衡常数时，要注意表达式的书写规则和温度对平衡常数的影响。
  • 对于化学平衡的移动问题，要应用勒夏特列原理，分析条件改变对平衡的影响。
  • 对于弱电解质的电离平衡问题，要理解电离常数的意义和影响因素，掌握pH的计算方法。
