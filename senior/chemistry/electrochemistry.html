电化学基础

1. 原电池

定义：将化学能转化为电能的装置

工作原理：通过氧化还原反应，电子从负极流向正极，产生电流

构成条件：

  • 两个活泼性不同的电极（正极和负极）
  • 电解质溶液
  • 形成闭合回路
  • 自发的氧化还原反应

电极反应：

  • 负极：发生氧化反应，失去电子，电子流出
  • 正极：发生还原反应，得到电子，电子流入

原电池的表示方法：（-）Zn | Zn²⁺（c₁）|| Cu²⁺（c₂）| Cu（+）

典型例题
例题1：基础题

下列装置中，能构成原电池的是（ ）

A. 两个铜电极插入稀硫酸中

B. 铜片和锌片插入稀硫酸中，用导线连接

C. 铜片和锌片插入硫酸铜溶液中，没有导线连接

D. 铜片和锌片插入酒精中，用导线连接

答案：B

解析：构成原电池需要两个活泼性不同的电极、电解质溶液、闭合回路和自发的氧化还原反应。A选项两个电极相同，不能构成原电池；C选项没有形成闭合回路；D选项酒精不是电解质溶液，不能构成原电池。

例题2：中等题

将锌片和铜片插入稀硫酸中，用导线连接，下列说法正确的是（ ）

A. 锌片是正极，发生还原反应

B. 铜片是负极，发生氧化反应

C. 电子从锌片流向铜片

D. 电流从锌片流向铜片

答案：C

解析：锌的活泼性比铜强，所以锌是负极，发生氧化反应；铜是正极，发生还原反应。电子从负极（锌片）流向正极（铜片），电流方向与电子流向相反。

例题3：提高题

某原电池总反应为：Zn + Cu²⁺ = Zn²⁺ + Cu，下列关于该原电池的说法正确的是（ ）

A. 负极材料为Cu，正极材料为Zn

B. 电解质溶液可以是稀盐酸

C. 正极反应式为：Cu²⁺ + 2e⁻ = Cu

D. 电子从正极流向负极

答案：C

解析：根据总反应，Zn失去电子，所以Zn是负极，Cu²⁺得到电子，所以正极反应式为Cu²⁺ + 2e⁻ = Cu。电解质溶液应含有Cu²⁺，如硫酸铜溶液，稀盐酸中没有Cu²⁺，不能构成该原电池。

常见错误分析
  • 错误1：认为所有金属都能作为原电池的电极。实际上，两个电极的活泼性必须不同。
  • 错误2：混淆电子流向和电流流向。电子从负极流向正极，电流从正极流向负极。
  • 错误3：忽略电解质溶液的作用。电解质溶液必须能导电，且参与或促进氧化还原反应。
解题技巧总结
  • 判断原电池的正负极：活泼金属作负极，不活泼金属或导电非金属作正极。
  • 书写电极反应式：负极发生氧化反应，正极发生还原反应，注意电荷守恒和质量守恒。
  • 分析原电池的工作原理：从氧化还原反应的角度理解电子转移过程。

2. 电解池

定义：将电能转化为化学能的装置

工作原理：通过外加电源，使电解质溶液或熔融电解质发生氧化还原反应

构成条件：

  • 直流电源
  • 两个电极（阳极和阴极）
  • 电解质溶液或熔融电解质
  • 形成闭合回路

电极反应：

  • 阳极：与电源正极相连，发生氧化反应，失去电子
  • 阴极：与电源负极相连，发生还原反应，得到电子

电解规律：

  • 阳离子放电顺序（从难到易）：K⁺ < Na⁺ < Ca²⁺ < Mg²⁺ < Al³⁺ < H⁺ < Cu²⁺ < Fe³⁺ < Ag⁺
  • 阴离子放电顺序（从难到易）：S²⁻ > I⁻ > Br⁻ > Cl⁻ > OH⁻ > 含氧酸根
典型例题
例题1：基础题

下列关于电解池的说法正确的是（ ）

A. 电解池的阳极发生还原反应

B. 电解池的阴极与电源正极相连

C. 电解池中电子从阳极流向阴极

D. 电解池是将电能转化为化学能的装置

答案：D

解析：电解池的阳极发生氧化反应，阴极发生还原反应；阳极与电源正极相连，阴极与电源负极相连；电子从电源负极流向阴极，从阳极流向电源正极。

例题2：中等题

用惰性电极电解饱和食盐水，下列说法正确的是（ ）

A. 阳极产生H₂

B. 阴极产生Cl₂

C. 溶液的pH增大

D. 总反应式为：2NaCl + 2H₂O 电解 2NaOH + H₂↑ + Cl₂↑

答案：CD

解析：用惰性电极电解饱和食盐水，阳极发生反应：2Cl⁻ - 2e⁻ = Cl₂↑，阴极发生反应：2H₂O + 2e⁻ = H₂↑ + 2OH⁻，总反应式为2NaCl + 2H₂O 电解 2NaOH + H₂↑ + Cl₂↑，溶液的pH增大。

例题3：提高题

用惰性电极电解硫酸铜溶液，当电路中通过0.5mol电子时，下列说法正确的是（ ）

A. 阴极析出16g Cu

B. 阳极析出0.5mol O₂

C. 溶液中Cu²⁺浓度减小

D. 溶液的pH减小

答案：AC

解析：用惰性电极电解硫酸铜溶液，阴极发生反应：Cu²⁺ + 2e⁻ = Cu，阳极发生反应：2H₂O - 4e⁻ = O₂↑ + 4H⁺。当电路中通过0.5mol电子时，阴极析出0.25mol Cu（16g），阳极析出0.125mol O₂。溶液中Cu²⁺浓度减小，H⁺浓度增大，pH减小。

常见错误分析
  • 错误1：混淆电解池的阴阳极与原电池的正负极。电解池的阳极与电源正极相连，阴极与电源负极相连。
  • 错误2：忽略电极材料的影响。惰性电极（如Pt、石墨）不参与反应，活性电极（如Cu、Fe）作阳极时会参与反应。
  • 错误3：错误判断放电顺序。阳离子放电顺序与金属活动性顺序相反，阴离子放电顺序与非金属性顺序有关。
解题技巧总结
  • 判断电解池的阴阳极：与电源正极相连的是阳极，与电源负极相连的是阴极。
  • 确定放电顺序：根据离子的氧化性和还原性强弱判断放电顺序。
  • 书写电极反应式：阳极发生氧化反应，阴极发生还原反应，注意电解质溶液的酸碱性。
  • 计算电解产物的量：根据电子守恒进行计算。

3. 金属的腐蚀与防护

金属腐蚀的定义：金属或合金与周围环境中的物质发生化学反应而腐蚀损耗的现象

金属腐蚀的分类：

  • 化学腐蚀：金属与非电解质直接发生化学反应而引起的腐蚀
  • 电化学腐蚀：金属与电解质溶液构成原电池，发生氧化还原反应而引起的腐蚀

电化学腐蚀的类型：

  • 析氢腐蚀：在酸性环境中，正极反应式为：2H⁺ + 2e⁻ = H₂↑
  • 吸氧腐蚀：在中性或弱酸性环境中，正极反应式为：O₂ + 2H₂O + 4e⁻ = 4OH⁻

金属的防护方法：

  • 改变金属内部结构：如制成不锈钢
  • 覆盖保护层：如涂油、喷漆、电镀等
  • 电化学防护：
    • 牺牲阳极的阴极保护法：将被保护金属与更活泼的金属连接
    • 外加电流的阴极保护法：将被保护金属与电源负极连接
典型例题
例题1：基础题

下列关于金属腐蚀的说法正确的是（ ）

A. 金属腐蚀的本质是金属原子失去电子被氧化

B. 金属腐蚀分为化学腐蚀和电化学腐蚀

C. 电化学腐蚀比化学腐蚀更普遍

D. 金属腐蚀都是有害的

答案：ABC

解析：金属腐蚀的本质是金属原子失去电子被氧化；金属腐蚀分为化学腐蚀和电化学腐蚀，其中电化学腐蚀更普遍；金属腐蚀在某些情况下也可以被利用，如金属的蚀刻。

例题2：中等题

下列金属防护方法中，属于电化学防护的是（ ）

A. 喷油漆

B. 电镀

C. 制成不锈钢

D. 牺牲阳极的阴极保护法

答案：D

解析：喷油漆和电镀属于覆盖保护层；制成不锈钢属于改变金属内部结构；牺牲阳极的阴极保护法属于电化学防护。

例题3：提高题

下列关于钢铁腐蚀的说法正确的是（ ）

A. 钢铁在潮湿的空气中发生的主要是析氢腐蚀

B. 钢铁在酸性环境中发生的主要是吸氧腐蚀

C. 钢铁腐蚀的负极反应式为：Fe - 2e⁻ = Fe²⁺

D. 牺牲阳极的阴极保护法中，牺牲的阳极是比钢铁更不活泼的金属

答案：C

解析：钢铁在潮湿的空气中发生的主要是吸氧腐蚀；在酸性环境中发生的主要是析氢腐蚀；钢铁腐蚀的负极反应式为Fe - 2e⁻ = Fe²⁺；牺牲阳极的阴极保护法中，牺牲的阳极是比钢铁更活泼的金属。

常见错误分析
  • 错误1：认为金属腐蚀只发生在潮湿的环境中。实际上，干燥的环境中也会发生化学腐蚀。
  • 错误2：混淆析氢腐蚀和吸氧腐蚀的条件。析氢腐蚀发生在酸性环境中，吸氧腐蚀发生在中性或弱酸性环境中。
  • 错误3：错误选择牺牲阳极的材料。牺牲阳极必须是比被保护金属更活泼的金属。
解题技巧总结
  • 判断金属腐蚀的类型：根据环境的酸碱性判断是析氢腐蚀还是吸氧腐蚀。
  • 选择金属防护方法：根据金属的用途和环境条件选择合适的防护方法。
  • 理解电化学防护的原理：牺牲阳极的阴极保护法利用原电池原理，外加电流的阴极保护法利用电解池原理。
