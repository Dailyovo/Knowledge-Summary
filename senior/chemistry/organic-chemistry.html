有机化学基础

1. 烃的衍生物

烃的衍生物的定义：烃分子中的氢原子被其他原子或原子团取代而生成的一系列化合物

官能团：决定有机化合物化学性质的原子或原子团

1.1 卤代烃

定义：烃分子中的氢原子被卤素原子取代而生成的化合物

通式：R-X（X为卤素原子，F、Cl、Br、I）

主要化学性质：

  • 水解反应（取代反应）：R-X + NaOH → R-OH + NaX（条件：水，加热）
  • 消去反应：R-CH₂-CH₂-X + NaOH → R-CH=CH₂ + NaX + H₂O（条件：醇，加热）
1.2 醇

定义：烃分子中的氢原子被羟基（-OH）取代而生成的化合物

通式：R-OH

主要化学性质：

  • 与活泼金属反应：2R-OH + 2Na → 2R-ONa + H₂↑
  • 氧化反应：
    • 燃烧：C₂H₅OH + 3O₂ → 2CO₂ + 3H₂O
    • 催化氧化：2R-CH₂-OH + O₂ → 2R-CHO + 2H₂O（条件：Cu或Ag，加热）
  • 消去反应：CH₃CH₂OH → CH₂=CH₂↑ + H₂O（条件：浓H₂SO₄，170℃）
  • 酯化反应：R-OH + R'-COOH → R'-COOR + H₂O（条件：浓H₂SO₄，加热）
1.3 酚

定义：羟基直接连接在苯环上的化合物

通式：Ar-OH（Ar为芳香烃基）

主要化学性质：

  • 弱酸性：C₆H₅OH + NaOH → C₆H₅ONa + H₂O
  • 取代反应：与浓溴水反应生成2,4,6-三溴苯酚白色沉淀
  • 显色反应：与FeCl₃溶液反应显紫色
1.4 醛

定义：分子中含有醛基（-CHO）的化合物

通式：R-CHO

主要化学性质：

  • 氧化反应：
    • 银镜反应：R-CHO + 2Ag(NH₃)₂OH → R-COONH₄ + 2Ag↓ + 3NH₃ + H₂O
    • 与新制Cu(OH)₂反应：R-CHO + 2Cu(OH)₂ + NaOH → R-COONa + Cu₂O↓ + 3H₂O
  • 还原反应：R-CHO + H₂ → R-CH₂-OH（条件：催化剂，加热）
1.5 羧酸

定义：分子中含有羧基（-COOH）的化合物

通式：R-COOH

主要化学性质：

  • 酸性：具有酸的通性，能与活泼金属、碱、盐等反应
  • 酯化反应：与醇反应生成酯
1.6 酯

定义：羧酸与醇发生酯化反应生成的化合物

通式：R-COOR'

主要化学性质：

  • 水解反应：
    • 酸性水解：R-COOR' + H₂O → R-COOH + R'-OH（条件：稀H₂SO₄，加热）
    • 碱性水解（皂化反应）：R-COOR' + NaOH → R-COONa + R'-OH
典型例题
例题1：基础题

下列物质中，属于醇类的是（ ）

A. C₆H₅OH

B. CH₃CH₂OH

C. CH₃COOH

D. CH₃CHO

答案：B

解析：醇是羟基连接在脂肪烃基上的化合物，A选项羟基直接连接在苯环上，属于酚类；C选项含有羧基，属于羧酸；D选项含有醛基，属于醛类。

例题2：中等题

下列反应中，属于消去反应的是（ ）

A. 乙醇与金属钠反应

B. 乙醇在铜催化下与氧气反应

C. 乙醇与浓硫酸共热到170℃

D. 乙醇与浓硫酸共热到140℃

答案：C

解析：消去反应是指有机化合物在一定条件下，从一个分子中脱去一个或几个小分子（如水、卤化氢等），而生成含不饱和键的化合物的反应。A选项属于置换反应；B选项属于氧化反应；C选项属于消去反应，生成乙烯；D选项属于取代反应，生成乙醚。

例题3：提高题

某有机物能发生银镜反应，与金属钠反应放出氢气，能与碳酸钠反应生成二氧化碳，则该有机物的结构简式可能是（ ）

A. CH₃CH₂OH

B. CH₃CHO

C. CH₃COOH

D. HOOC-CHO

答案：D

解析：能发生银镜反应说明含有醛基（-CHO），能与金属钠反应放出氢气说明含有羟基（-OH）或羧基（-COOH），能与碳酸钠反应生成二氧化碳说明含有羧基。综合以上条件，只有D选项同时含有醛基和羧基。

常见错误分析
  • 错误1：混淆醇和酚的概念。醇的羟基连接在脂肪烃基上，酚的羟基直接连接在苯环上。
  • 错误2：忽略消去反应的条件。醇发生消去反应需要浓硫酸作催化剂，加热到170℃，且与羟基相连的碳原子的邻位碳原子上必须有氢原子。
  • 错误3：错误判断有机反应类型。如将酯化反应误认为是加成反应，实际上酯化反应属于取代反应。
解题技巧总结
  • 根据官能团判断有机物的类别和性质：不同的官能团决定了有机物的不同化学性质。
  • 掌握有机反应的条件：相同的反应物在不同条件下可能发生不同的反应，如乙醇在浓硫酸作用下，170℃发生消去反应，140℃发生取代反应。
  • 理解有机反应的机理：从化学键的断裂和形成角度理解有机反应的本质。

2. 有机反应类型

有机反应的主要类型：

2.1 取代反应

定义：有机物分子里的某些原子或原子团被其他原子或原子团所代替的反应

常见类型：

  • 卤代反应：如甲烷与氯气在光照条件下反应
  • 硝化反应：如苯与浓硝酸在浓硫酸催化下反应
  • 酯化反应：如乙醇与乙酸在浓硫酸催化下反应
  • 水解反应：如卤代烃、酯、多糖、蛋白质的水解
2.2 加成反应

定义：有机物分子中双键（或三键）两端的碳原子与其他原子或原子团直接结合生成新的化合物的反应

常见类型：

  • 与氢气加成：如烯烃、炔烃、苯及其同系物与氢气的加成
  • 与卤素加成：如烯烃、炔烃与卤素单质的加成
  • 与卤化氢加成：如烯烃、炔烃与卤化氢的加成
  • 与水加成：如烯烃与水的加成
2.3 消去反应

定义：有机物在一定条件下，从一个分子中脱去一个或几个小分子（如水、卤化氢等），而生成含不饱和键的化合物的反应

常见类型：

  • 醇的消去反应：如乙醇在浓硫酸作用下生成乙烯
  • 卤代烃的消去反应：如溴乙烷在氢氧化钠的醇溶液中生成乙烯
2.4 氧化反应

定义：有机物分子中失去氢原子或得到氧原子的反应

常见类型：

  • 燃烧反应：如有机物的完全燃烧
  • 催化氧化：如乙醇在铜催化下生成乙醛
  • 与氧化剂反应：如乙醛与银氨溶液、新制氢氧化铜的反应
2.5 还原反应

定义：有机物分子中得到氢原子或失去氧原子的反应

常见类型：

  • 与氢气加成：如醛、酮与氢气加成生成醇
2.6 聚合反应

定义：由相对分子质量小的化合物分子互相结合成相对分子质量大的高分子化合物的反应

常见类型：

  • 加聚反应：由不饱和的单体加成聚合生成高分子化合物的反应，如乙烯聚合生成聚乙烯
  • 缩聚反应：由单体通过缩合反应生成高分子化合物，同时生成小分子（如水、氨等）的反应，如对苯二甲酸与乙二醇缩聚生成聚酯纤维
典型例题
例题1：基础题

下列反应中，属于加成反应的是（ ）

A. 甲烷与氯气在光照条件下反应

B. 乙烯与溴水反应

C. 乙醇与浓硫酸共热到170℃

D. 乙酸与乙醇在浓硫酸催化下反应

答案：B

解析：加成反应是指有机物分子中双键或三键两端的碳原子与其他原子或原子团直接结合生成新化合物的反应。A选项属于取代反应；C选项属于消去反应；D选项属于取代反应（酯化反应）。

例题2：中等题

下列反应中，属于氧化反应的是（ ）

A. 乙烯与氢气反应生成乙烷

B. 乙醇在铜催化下与氧气反应生成乙醛

C. 苯与浓硝酸反应生成硝基苯

D. 乙酸与乙醇反应生成乙酸乙酯

答案：B

解析：氧化反应是指有机物分子中失去氢原子或得到氧原子的反应。A选项属于还原反应；C选项属于取代反应；D选项属于取代反应（酯化反应）。

例题3：提高题

下列反应中，属于缩聚反应的是（ ）

A. 乙烯聚合生成聚乙烯

B. 氯乙烯聚合生成聚氯乙烯

C. 对苯二甲酸与乙二醇缩聚生成聚酯纤维

D. 苯乙烯聚合生成聚苯乙烯

答案：C

解析：缩聚反应是指由单体通过缩合反应生成高分子化合物，同时生成小分子的反应。A、B、D选项都属于加聚反应，只有C选项属于缩聚反应，生成聚酯纤维的同时生成水。

常见错误分析
  • 错误1：混淆取代反应和加成反应的概念。取代反应是原子或原子团的替换，加成反应是双键或三键的打开，原子或原子团的添加。
  • 错误2：错误判断氧化反应和还原反应。有机物的氧化反应是失去氢原子或得到氧原子，还原反应是得到氢原子或失去氧原子。
  • 错误3：混淆加聚反应和缩聚反应。加聚反应只有高分子化合物生成，缩聚反应除了高分子化合物外，还生成小分子。
解题技巧总结
  • 根据反应物和生成物的结构变化判断反应类型：分析反应前后有机物分子结构的变化，确定发生的反应类型。
  • 掌握各类反应的典型代表：熟悉各类有机反应的典型实例，如取代反应的典型代表是甲烷的卤代反应，加成反应的典型代表是乙烯与溴水的反应。
  • 注意反应条件的重要性：相同的反应物在不同条件下可能发生不同类型的反应，反应条件是判断反应类型的重要依据。

3. 有机合成

有机合成的定义：利用简单、易得的原料，通过有机反应，生成具有特定结构和功能的有机化合物的过程

有机合成的基本任务：

  • 构建碳骨架：增长或缩短碳链，形成碳环或开环
  • 引入官能团：在碳骨架上引入所需的官能团
  • 官能团的转化：将一种官能团转化为另一种官能团
3.1 官能团的引入

常见官能团的引入方法：

  • 引入碳碳双键：
    • 醇的消去反应：如乙醇→乙烯
    • 卤代烃的消去反应：如溴乙烷→乙烯
    • 炔烃的不完全加成：如乙炔→乙烯
  • 引入卤原子：
    • 烷烃的卤代反应：如甲烷→一氯甲烷
    • 烯烃、炔烃的加成反应：如乙烯→1,2-二溴乙烷
    • 醇的取代反应：如乙醇→溴乙烷
  • 引入羟基：
    • 烯烃的加成反应：如乙烯→乙醇
    • 卤代烃的水解反应：如溴乙烷→乙醇
    • 醛、酮的还原反应：如乙醛→乙醇
    • 酯的水解反应：如乙酸乙酯→乙醇和乙酸
  • 引入醛基：
    • 醇的催化氧化：如乙醇→乙醛
    • 烯烃的氧化反应：如乙烯→乙醛
  • 引入羧基：
    • 醛的氧化反应：如乙醛→乙酸
    • 酯的水解反应：如乙酸乙酯→乙酸和乙醇
    • 苯的同系物的氧化反应：如甲苯→苯甲酸
3.2 官能团的转化

常见官能团的转化关系：

卤代烃 ↔ 醇 ↔ 醛 ↔ 羧酸 ↔ 酯

转化示例：

  • 卤代烃→醇：水解反应
  • 醇→卤代烃：取代反应
  • 醇→醛：氧化反应
  • 醛→醇：还原反应
  • 醛→羧酸：氧化反应
  • 羧酸→酯：酯化反应
  • 酯→羧酸：水解反应
3.3 有机合成路线的设计

设计思路：

  • 正向合成法：从原料出发，逐步合成目标化合物
  • 逆向合成法：从目标化合物出发，逐步推导出原料

设计原则：

  • 原料廉价易得
  • 步骤简单，产率高
  • 条件温和，操作安全
  • 尽量减少对环境的污染
典型例题
例题1：基础题

下列物质中，能通过乙烯加成反应直接得到的是（ ）

A. CH₃CH₃

B. CH₃CH₂Cl

C. CH₃CH₂OH

D. 以上都可以

答案：D

解析：乙烯与氢气加成生成乙烷，与氯化氢加成生成氯乙烷，与水加成生成乙醇，所以以上物质都可以通过乙烯加成反应直接得到。

例题2：中等题

以乙醇为原料，合成乙酸乙酯的路线中，需要经过的反应类型是（ ）

A. 氧化反应、酯化反应

B. 还原反应、酯化反应

C. 消去反应、加成反应

D. 水解反应、酯化反应

答案：A

解析：乙醇合成乙酸乙酯的路线是：乙醇→乙醛→乙酸→乙酸乙酯。其中乙醇→乙醛是氧化反应，乙酸→乙酸乙酯是酯化反应。

例题3：提高题

设计以乙烯为原料合成乙酸乙酯的路线，写出各步反应的化学方程式。

答案：

1. 乙烯与水加成生成乙醇：CH₂=CH₂ + H₂O → CH₃CH₂OH（条件：催化剂，加热，加压）

2. 乙醇氧化生成乙醛：2CH₃CH₂OH + O₂ → 2CH₃CHO + 2H₂O（条件：Cu，加热）

3. 乙醛氧化生成乙酸：2CH₃CHO + O₂ → 2CH₃COOH（条件：催化剂，加热）

4. 乙醇与乙酸酯化生成乙酸乙酯：CH₃CH₂OH + CH₃COOH → CH₃COOCH₂CH₃ + H₂O（条件：浓H₂SO₄，加热）

解析：以乙烯为原料合成乙酸乙酯，需要先构建碳骨架，然后引入官能团，并进行官能团的转化。首先乙烯与水加成生成乙醇，然后乙醇氧化生成乙醛，乙醛再氧化生成乙酸，最后乙醇与乙酸酯化生成乙酸乙酯。

常见错误分析
  • 错误1：有机合成路线设计不合理，步骤过多，产率低。
  • 错误2：官能团引入顺序错误，导致副反应发生。
  • 错误3：忽略反应条件，导致反应无法进行。
解题技巧总结
  • 熟练掌握官能团的引入和转化方法：这是有机合成的基础，只有掌握了各种官能团的引入和转化方法，才能设计出合理的合成路线。
  • 学会逆向合成法：从目标化合物出发，逐步推导出原料，这是设计复杂有机合成路线的常用方法。
  • 注意反应条件的选择：不同的反应条件会影响反应的方向和产率，选择合适的反应条件是有机合成的关键。
  • 考虑原子经济性：在设计合成路线时，应尽量选择原子利用率高的反应，减少废弃物的产生。
