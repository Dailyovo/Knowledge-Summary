复数

概念阐释：复数是实数的扩展，引入了虚数单位i，使得方程x² = -1有解。复数在数学、物理、工程等领域有着广泛的应用，特别是在电路分析、信号处理、量子力学等方面。

1. 复数的基本概念

虚数单位：i，满足i² = -1。

复数的定义：形如a + bi（a, b ∈ R）的数叫做复数，其中a叫做复数的实部，记作Re(z)；b叫做复数的虚部，记作Im(z)。

复数的分类：

  • 实数：当b = 0时，复数a + bi就是实数a
  • 虚数：当b ≠ 0时，复数a + bi叫做虚数
  • 纯虚数：当a = 0且b ≠ 0时，复数$a + bi$叫做纯虚数，即$bi$

复数相等：两个复数a + bi和c + di相等的充要条件是a = c且b = d。

复数的模：复数z = a + bi的模（或绝对值）记作|z|，定义为： $$|z| = \sqrt{a² + b²}$$

共轭复数：复数z = a + bi的共轭复数记作\overline{z}，定义为： $$\overline{z} = a - bi$$

2. 复数的几何表示

复平面：建立了直角坐标系来表示复数的平面叫做复平面，x轴叫做实轴，y轴叫做虚轴。

复数的几何表示：复数$z = a + bi$可以用复平面内的点$Z(a, b)$表示，也可以用向量$\overrightarrow{OZ}$表示，其中O是坐标原点。

复数的模的几何意义：复数$z = a + bi$的模$|z|$表示复平面内点$Z(a, b)$到原点O的距离，即向量$\overrightarrow{OZ}$的长度。

共轭复数的几何意义：复数$z = a + bi$的共轭复数$\overline{z}$表示复平面内点$Z(a, b)$关于实轴的对称点$\overline{Z}(a, -b)$。

复数的辐角：设复数$z = a + bi$对应的向量为$\overrightarrow{OZ}$，则向量$\overrightarrow{OZ}$与实轴正方向的夹角$θ$叫做复数$z$的辐角，记作$Arg(z)$。辐角的主值记作$arg(z)$，范围是$[0, 2π)$。

复数的三角形式：设复数z = a + bi的模为r，辐角为θ，则复数z可以表示为： $$z = r(cosθ + isinθ)$$其中$r = |z| = \sqrt{a² + b²}$，$cosθ = a/r$，$sinθ = b/r$。

3. 复数的运算

复数的加法：设$z₁ = a + bi$，$z₂ = c + di$，则： $$z₁ + z₂ = (a + c) + (b + d)i$$

复数的减法：设$z₁ = a + bi$，$z₂ = c + di$，则： $$z₁ - z₂ = (a - c) + (b - d)i$$

复数的乘法：

  • 代数形式：设$z₁ = a + bi$，$z₂ = c + di$，则： $$z₁z₂ = (ac - bd) + (ad + bc)i$$
  • 三角形式：设$z₁ = r₁(cosθ₁ + isinθ₁)$，$z₂ = r₂(cosθ₂ + isinθ₂)$，则： $$z₁z₂ = r₁r₂[cos(θ₁ + θ₂) + isin(θ₁ + θ₂)]$$

复数的除法：

  • 代数形式：设$z₁ = a + bi$，$z₂ = c + di$（$c + di ≠ 0$），则： $$\frac{z₁}{z₂} = \frac{(a + bi)(c - di)}{(c + di)(c - di)} = \frac{ac + bd}{c² + d²} + \frac{bc - ad}{c² + d²}i$$
  • 三角形式：设$z₁ = r₁(cosθ₁ + isinθ₁)$，$z₂ = r₂(cosθ₂ + isinθ₂)$（$r₂ ≠ 0$），则： $$\frac{z₁}{z₂} = \frac{r₁}{r₂}[cos(θ₁ - θ₂) + isin(θ₁ - θ₂)]$$

复数的乘方：（棣莫弗定理）设$z = r(cosθ + isinθ)$，则： $$zⁿ = rⁿ(cosnθ + isinnθ)$$（n ∈ N*）

复数的开方：设$z = r(cosθ + isinθ)$，则$z$的n次方根为： $$\sqrt[n]{r}[cos\frac{θ + 2kπ}{n} + isin\frac{θ + 2kπ}{n}]$$（k = 0, 1, 2, …, n - 1）

4. 复数的性质

共轭复数的性质：

  • $\overline{\overline{z}} = z$
  • $\overline{z₁ + z₂} = \overline{z₁} + \overline{z₂}$
  • $\overline{z₁ - z₂} = \overline{z₁} - \overline{z₂}$
  • $\overline{z₁z₂} = \overline{z₁}·\overline{z₂}$
  • $\overline{\frac{z₁}{z₂}} = \frac{\overline{z₁}}{\overline{z₂}}$（z₂ ≠ 0）
  • $z·\overline{z} = |z|²$
  • $Re(z) = \frac{z + \overline{z}}{2}$，$Im(z) = \frac{z - \overline{z}}{2i}$

复数模的性质：

  • $|z| = |\overline{z}|$
  • $|z₁z₂| = |z₁||z₂|$
  • $|\frac{z₁}{z₂}| = \frac{|z₁|}{|z₂|}$（z₂ ≠ 0）
  • $|z₁ + z₂| ≤ |z₁| + |z₂|$（三角不等式）
  • $|z₁ - z₂| ≥ ||z₁| - |z₂||$
  • $|z|² = z·\overline{z}$
典型例题

例1（基础题）：已知复数z = (3 + i)(1 - 2i)，求z的实部、虚部、模和共轭复数。

解析：先计算z的代数形式， $$z = (3 + i)(1 - 2i) = 3×1 + 3×(-2i) + i×1 + i×(-2i) = 3 - 6i + i - 2i² = 3 - 5i - 2×(-1) = 5 - 5i$$ - 实部：$Re(z) = 5$ - 虚部：$Im(z) = -5$ - 模：$|z| = \sqrt{5² + (-5)²} = \sqrt{50} = 5\sqrt{2}$ - 共轭复数：$\overline{z} = 5 + 5i$

例2（中档题）：已知复数z满足$|z| = 1$，且$z + \frac{1}{z} = 1$，求z。

解析：设z = a + bi（a, b ∈ R），则$|z| = \sqrt{a² + b²} = 1$，即a² + b² = 1。 又$z + \frac{1}{z} = z + \frac{\overline{z}}{|z|²} = z + \overline{z} = 2a = 1$，所以a = 1/2。 代入$a² + b² = 1$，得$(1/2)² + b² = 1$，解得$b = ±\sqrt{3}/2$。 因此，$z = 1/2 ± (\sqrt{3}/2)i$。

例3（提高题）：已知复数$z₁ = cosθ + isinθ$，$z₂ = cosφ + isinφ$，求证： $$z₁z₂ + \frac{z₁}{z₂} = 2cos(θ - φ)$$

解析： $$z₁z₂ = (cosθ + isinθ)(cosφ + isinφ) = cos(θ + φ) + isin(θ + φ)$$ $$\frac{z₁}{z₂} = \frac{cosθ + isinθ}{cosφ + isinφ} = cos(θ - φ) + isin(θ - φ)$$ 所以， $$z₁z₂ + \frac{z₁}{z₂} = [cos(θ + φ) + cos(θ - φ)] + i[sin(θ + φ) + sin(θ - φ)]$$ 利用三角恒等式： $$cosA + cosB = 2cos\frac{A + B}{2}cos\frac{A - B}{2}$$ $$sinA + sinB = 2sin\frac{A + B}{2}cos\frac{A - B}{2}$$ 代入A = θ + φ，B = θ - φ，得： $$z₁z₂ + \frac{z₁}{z₂} = 2cosθcosφ + i·2sinθcosφ = 2cosφ(cosθ + isinθ)$$ 但这与要证明的结果不符，说明我们的方法有误。 重新计算： $$z₁z₂ + \frac{z₁}{z₂} = z₁(z₂ + \frac{1}{z₂}) = z₁(2cosφ) = 2cosφ(cosθ + isinθ)$$ 哦，原来题目可能有误，或者我们理解错了。再仔细看题目，应该是z₁\overline{z₂} + \overline{z₁}z₂ = 2cos(θ - φ)。 证明： $$z₁\overline{z₂} = (cosθ + isinθ)(cosφ - isinφ) = cos(θ - φ) + isin(θ - φ)$$ $$\overline{z₁}z₂ = (cosθ - isinθ)(cosφ + isinφ) = cos(θ - φ) - isin(θ - φ)$$ 所以， $$z₁\overline{z₂} + \overline{z₁}z₂ = 2cos(θ - φ)$$

常见错误分析
  • 错误1：误将复数的虚部当作$bi$，而实际上虚部是$b$。
    正确做法：复数$a + bi$的实部是$a$，虚部是$b$，不是$bi$。
  • 错误2：在计算复数的除法时，忘记乘以共轭复数。
    正确做法：计算复数除法时，分子分母同乘以分母的共轭复数，将分母实数化。
  • 错误3：在使用棣莫弗定理时，误将辐角的n次方当作n倍。
    正确做法：棣莫弗定理中，辐角是乘以n，不是n次方，即$zⁿ = rⁿ(cosnθ + isinnθ)$。
  • 错误4：误将复数的模的平方当作$a² + (bi)²$，而实际上是$a² + b²$。
    正确做法：复数z = a + bi的模的平方是$|z|² = a² + b²$，不是$a² + (bi)²$。
解题技巧总结
  • 对于复数的运算，通常先将复数化为代数形式，再进行计算。
  • 利用共轭复数的性质可以简化复数的除法运算，将分母实数化。
  • 复数的模和共轭复数的性质是解决复数问题的重要工具，要熟练掌握。
  • 复数的几何意义（复平面、向量表示）可以帮助理解复数的运算和性质。
  • 对于涉及复数模的问题，常利用$|z|² = z·\overline{z}$进行转化。
