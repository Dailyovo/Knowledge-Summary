电场与磁场

1. 电场强度

定义式：$ E = \frac{F}{q} $（适用于任何电场）

点电荷的电场强度：$ E = k\frac{Q}{r^2} $（k为静电力常量，$ k = 9.0 \times 10^9 N\cdot m^2/C^2 $）

匀强电场的电场强度：$ E = \frac{U}{d} $（U为两点间的电势差，d为沿电场方向的距离）

2. 电势与电势差

电势：$ \varphi = \frac{E_p}{q} $（E_p为电势能）

电势差：$ U_{AB} = \varphi_A - \varphi_B = \frac{W_{AB}}{q} $（W_AB为电场力做的功）

3. 磁场基本概念

磁感应强度：$ B = \frac{F}{IL} $（F为安培力，I为电流，L为导线长度）

磁通量：$ \Phi = BScos \theta $（S为面积，θ为B与S的夹角）

4. 安培力与洛伦兹力

安培力：$ F = BILsin \theta $（θ为B与I的夹角）

洛伦兹力：$ F = qvBsin \theta $（θ为B与v的夹角）

洛伦兹力提供向心力：$ qvB = m\frac{v^2}{r} $ ⇒ $ r = \frac{mv}{qB} $，周期$ T = \frac{2\pi m}{qB} $

典型例题

例1（基础题）：在电场强度为2×10³ N/C的匀强电场中，有一个电荷量为5×10⁻⁶ C的点电荷，求该电荷所受的电场力大小。

解析：根据电场强度的定义式$ E = \frac{F}{q} $，可得电场力$ F = Eq = 2×10³ N/C × 5×10⁻⁶ C = 0.01 N $。

例2（中档题）：一个质量为m、电荷量为q的带电粒子，以速度v垂直进入磁感应强度为B的匀强磁场中，求粒子做圆周运动的半径和周期。

解析：洛伦兹力提供向心力，即$ qvB = m\frac{v^2}{r} $，解得半径$ r = \frac{mv}{qB} $；周期$ T = \frac{2\pi r}{v} = \frac{2\pi m}{qB} $。

例3（提高题）：在如图所示的平行板电容器中，两板间距离为d，板长为L，两板间电势差为U，一个质量为m、电荷量为q的带电粒子以初速度v₀沿水平方向射入电场，求粒子离开电场时的偏转距离和偏转角。

解析：粒子在电场中做类平抛运动，水平方向做匀速直线运动，运动时间$ t = \frac{L}{v₀} $；竖直方向做匀加速直线运动，加速度$ a = \frac{F}{m} = \frac{qE}{m} = \frac{qU}{md} $。偏转距离$ y = \frac{1}{2}at² = \frac{1}{2}×\frac{qU}{md}×(\frac{L}{v₀})² = \frac{qUL²}{2mdv₀²} $。偏转角θ的正切值$ tanθ = \frac{v_y}{v₀} = \frac{at}{v₀} = \frac{qUL}{mdv₀²} $。

常见错误分析
  • 错误1：混淆电场强度的定义式和点电荷电场强度的决定式。
    正确做法：$ E = \frac{F}{q} $是定义式，适用于任何电场；$ E = k\frac{Q}{r^2} $是决定式，只适用于真空中的点电荷电场。
  • 错误2：在计算洛伦兹力时，忽略了速度方向与磁场方向的夹角。
    正确做法：洛伦兹力公式$ F = qvBsinθ $中的θ是速度方向与磁场方向的夹角，当v与B垂直时，F = qvB；当v与B平行时，F = 0。
  • 错误3：认为安培力和洛伦兹力的方向与磁场方向相同。
    正确做法：安培力和洛伦兹力的方向都垂直于磁场方向和电流/速度方向，可用左手定则判断。
解题技巧总结
  • 对于电场问题，要区分电场强度的定义式和决定式，理解电场线的性质和电势的分布规律。
  • 对于磁场问题，要掌握安培力和洛伦兹力的计算方法，理解左手定则的应用。
  • 对于带电粒子在磁场中的运动问题，要抓住洛伦兹力提供向心力这一关键点，结合圆周运动的公式求解。
  • 对于带电粒子在复合场中的运动问题，要分析粒子的受力情况，结合牛顿运动定律和运动学公式求解。
