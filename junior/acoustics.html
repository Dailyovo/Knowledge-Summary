声学学习

声音的产生与传播

声音的产生

声音的产生条件：声音是由物体的振动产生的。振动停止，发声也停止。

声源：正在发声的物体叫做声源。固体、液体、气体都可以作为声源。

典型实例：

  • 人说话时，声带振动发声
  • 鼓面振动发声
  • 水流撞击石头振动发声
  • 空气振动产生风声

声音的传播

传播条件：声音的传播需要介质，真空不能传声。

介质：能够传播声音的物质叫做介质。固体、液体、气体都可以作为传播声音的介质。

传播形式：声音以声波的形式传播。声波是一种机械波，具有能量。

声速

声速：声音在介质中传播的速度叫做声速。

影响因素：

  • 介质的种类：一般情况下，声音在固体中传播最快，液体中次之，气体中最慢。
  • 介质的温度：同一介质中，温度越高，声速越快。

数值：15℃时，声音在空气中的传播速度约为340m/s。

回声

回声：声音在传播过程中遇到障碍物被反射回来的现象叫做回声。

听到回声的条件：回声到达人耳的时间比原声晚0.1秒以上，人耳才能把回声和原声区分开。

应用：利用回声可以测量距离，如声呐探测、超声测距等。

公式：距离s = v × t / 2，其中v是声速，t是从发声到听到回声的时间。

声音的特性

音调

音调：声音的高低叫做音调。

影响因素：音调由发声体振动的频率决定。频率越高，音调越高；频率越低，音调越低。

频率：物体每秒内振动的次数叫做频率，单位是赫兹（Hz）。

人耳的听觉范围：人耳能听到的声音频率范围是20Hz～20000Hz。高于20000Hz的声音叫做超声波，低于20Hz的声音叫做次声波。

响度

响度：声音的强弱叫做响度。

影响因素：

  • 振幅：发声体振动的幅度叫做振幅。振幅越大，响度越大；振幅越小，响度越小。
  • 距离：距离发声体越近，响度越大；距离发声体越远，响度越小。

单位：响度的单位是分贝（dB）。0dB是人耳能听到的最微弱的声音，30dB～40dB是较理想的安静环境。

音色

音色：声音的品质和特色叫做音色。

影响因素：音色由发声体的材料、结构和发声方式决定。不同的发声体，即使音调和响度相同，音色也不同。

应用：根据音色可以辨别不同的发声体，如不同人的声音、不同乐器的声音等。

声音特性的对比

特性 概念 影响因素 描述词语
音调 声音的高低 频率 高低、尖锐、低沉
响度 声音的强弱 振幅、距离 大小、响亮、微弱
音色 声音的品质 材料、结构 悦耳、难听、不同乐器

噪声的危害与控制

噪声的定义

物理定义：从物理学角度看，噪声是发声体做无规则振动时发出的声音。

环保定义：从环境保护角度看，凡是妨碍人们正常休息、学习和工作的声音，以及对人们要听的声音产生干扰的声音，都属于噪声。

噪声的等级和危害

等级划分：噪声的等级用分贝（dB）表示。

危害：

  • 0dB：人耳能听到的最微弱的声音
  • 30dB～40dB：较理想的安静环境
  • 70dB：会影响人们的正常学习和工作
  • 90dB：会损坏听力，影响身体健康
  • 150dB：会导致鼓膜破裂，双耳失聪

噪声的控制

控制途径：

  • 在声源处减弱：防止噪声产生，如安装消声器、禁止鸣笛等。
  • 在传播过程中减弱：阻断噪声传播，如设置隔音墙、植树造林等。
  • 在人耳处减弱：防止噪声进入人耳，如戴耳塞、耳罩等。

声的利用

声音可以传递信息

应用实例：

  • 回声定位：声呐探测、蝙蝠导航
  • 医疗诊断：B超检查、听诊器
  • 工业检测：超声波探伤、超声检测
  • 生活应用：语音通信、广播、雷达

声音可以传递能量

应用实例：

  • 超声波清洗：清洗精密仪器、眼镜
  • 超声波碎石：击碎体内结石
  • 超声波焊接：焊接塑料、金属
  • 声波灭火：利用声波振动熄灭火灾

典型例题

例题1：基础题

关于声音的产生和传播，下列说法正确的是（ ）

A. 声音可以在真空中传播

B. 声音是由物体的振动产生的

C. 声音的传播速度与介质的温度无关

D. 声音在固体中传播速度比在空气中慢

答案：B

解析：声音的传播需要介质，真空不能传声，故A错误；声音是由物体的振动产生的，故B正确；声音的传播速度与介质的温度有关，温度越高，传播速度越快，故C错误；声音在固体中传播速度比在空气中快，故D错误。

例题2：中等题

在长为1000m的一根铁管两端有甲、乙两个同学，若乙同学在长铁管的一端敲一下铁管，甲同学把耳朵贴在长铁管的另一端听到两次声音的时间间隔为2.75s，求声音在铁中的传播速度。（已知声音在空气中的传播速度是340m/s）

答案：5200m/s

解析：设声音在铁中的传播速度为v铁，根据题意得：
声音在空气中的传播时间t1 = s / v空 = 1000m / 340m/s ≈ 2.94s
声音在铁中的传播时间t2 = s / v铁 = 1000m / v铁
时间间隔Δt = t1 - t2 = 2.75s
即2.94s - 1000m / v铁 = 2.75s
解得v铁 = 1000m / (2.94s - 2.75s) ≈ 5200m/s

例题3：提高题

一辆汽车以10m/s的速度向一座高山驶去，司机在某处鸣笛，鸣笛后3s听到回声，求汽车鸣笛处到高山的距离。（已知声音在空气中的传播速度是340m/s）

答案：525m

解析：设汽车鸣笛处到高山的距离为s，根据题意得：
在3s内，汽车行驶的距离s车 = v车 × t = 10m/s × 3s = 30m
在3s内，声音传播的距离s声 = v声 × t = 340m/s × 3s = 1020m
根据题意，s车 + 2s = s声
即30m + 2s = 1020m
解得s = (1020m - 30m) / 2 = 525m

实验探究

探究声音的产生条件

实验目的：探究声音产生的条件

实验器材：音叉、乒乓球、细线、铁架台、小锤

实验步骤：

  1. 将乒乓球用细线悬挂在铁架台上，调整乒乓球的位置，使其刚好能接触到静止的音叉
  2. 用小锤敲击音叉，观察乒乓球的运动情况
  3. 停止敲击音叉，观察乒乓球的运动情况

实验现象：敲击音叉时，音叉发声，乒乓球被弹开；停止敲击后，音叉停止发声，乒乓球停止运动。

实验结论：声音是由物体的振动产生的。

探究影响音调的因素

实验目的：探究影响音调的因素

实验器材：钢尺、桌面

实验步骤：

  1. 将钢尺的一端紧压在桌面上，另一端伸出桌面
  2. 用相同的力拨动钢尺，改变钢尺伸出桌面的长度，观察钢尺振动的快慢和发出声音的音调
  3. 记录不同伸出长度下的振动快慢和音调

实验现象：钢尺伸出桌面越短，振动越快，音调越高；钢尺伸出桌面越长，振动越慢，音调越低。

实验结论：音调由发声体振动的频率决定，频率越高，音调越高。

拓展阅读

超声波的应用

超声波是频率高于20000Hz的声波，具有方向性好、穿透能力强、易于获得较集中的声能等特点，在生产、医疗、科学研究等方面有广泛的应用：

  • 医疗领域：B超检查、超声波碎石、超声波清洗
  • 工业领域：超声波探伤、超声波焊接、超声波清洗
  • 农业领域：超声波育种、超声波灭虫
  • 军事领域：声呐探测、水下通信

次声波的危害与应用

次声波是频率低于20Hz的声波，具有传播距离远、穿透能力强等特点：

  • 危害：次声波会对人体造成危害，如引起头痛、恶心、呕吐等症状，严重时甚至会导致死亡。
  • 应用：次声波可以用于预测自然灾害（如地震、海啸）、监测核爆炸等。

实践活动

自制乐器

活动目标：学会自制简单的乐器，了解声音的特性

活动材料：玻璃杯、水、筷子

活动步骤：

  1. 准备5个相同的玻璃杯，分别装入不同量的水
  2. 用筷子敲击玻璃杯，听声音的音调
  3. 调整每个玻璃杯中水的量，使它们能发出不同音调的声音
  4. 尝试用自制的乐器演奏简单的乐曲

思考问题：为什么不同水量的玻璃杯发出的声音音调不同？

测量声音的速度

活动目标：学会测量声音的速度

活动材料：秒表、尺子

活动步骤：

  1. 选择一个空旷的场地，测量出一段距离（如100m）
  2. 一名同学在一端发出声音信号（如敲击木板），另一名同学在另一端用秒表测量从看到信号到听到声音的时间
  3. 根据公式v = s / t计算声音的速度
  4. 重复测量多次，取平均值，提高测量的准确性

思考问题：测量结果与标准声速有差异的原因是什么？
